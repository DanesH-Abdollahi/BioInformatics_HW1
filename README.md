# Semi-Global alignment

#### Some Examples :
Input Format :
```
>sequence1
>sequence2
```
Output Format :
```
Alignment Score : int
Alignment :
    sequence1
    sequence2
```

---
Input 1 :
```
HEAGAWGHE
PAWHEA
```
Output 1 :
```
20
HEAGAWGHE-
---PAW-HEA
```
---

---
Input 2 :
```
AAAAA
AA
```
Output 2 :
```
4
AAAAA
---AA
AAAAA
--AA-
AAAAA
-AA--
AAAAA
AA---
```
